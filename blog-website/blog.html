JARVIS: An AI Chatbot

JARVIS is a python-powered audio-based chatbot which receives user input through the browser, sends it to a Flask backend to be processed, and then returns the response to the user using text-to-speech. This project is still very buggy, but you can view the demo at this render link (please note that it may take some time to boot)!
